## 🔧 COMMANDES UTILES

### Charger Hermes-3.2-3B au lieu de 8B
```bash
docker run -d \
  --gpus=all \
  -v "d:/llm_models:/models:ro" \
  -p 8081:8080 \
  --name hermes-3b-light \
  ghcr.io/ggml-org/llama.cpp:full-cuda \
  /app/tools.sh --server \
  -m /models/Hermes-3-Llama-3.2-3B-Q4_K_M.gguf \
  --port 8080 --n-gpu-layers 20
```

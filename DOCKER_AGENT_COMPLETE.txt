## ⚡ TROUBLESHOOTING

### "LLM still loading"
```bash
# Check status
curl http://localhost:8080/health
# Expected: {"status":"ok"} after 1-2 minutes
# If longer, check logs:
docker logs llm-server-presparc
```

### "Agent not connecting to LLM"
```bash
# Verify network
docker network inspect presparc-net
# Should show both containers connected
```

### "Notion integration failing"
```bash
# Check logs
docker logs presparc-agent
# Look for "NOTION_API_KEY" errors
# Verify key in .env
```

### "Out of memory"
```bash
# Check resource usage
docker stats
# GPU memory limit: 12GB (configurable in docker-compose-agent-simple.yml line 31)
# If needed, reduce --n-gpu-layers or model size
```

================================================================================
                        TROUBLESHOOTING
================================================================================

"Hermes not responding"
  → Check LLM is running: docker logs llm-server-presparc
  → Check port: curl http://localhost:8080/health
  → Should return: {"status":"ok"}

"Notion connection failing"
  → Verify API key in .env: NOTION_API_KEY should NOT be empty
  → Check database ID is set: NOTION_DATABASE_ID
  → Try "Sync Notion database" command in Hermes

"Response is slow"
  → Normal for first query (LLM warming up)
  → Typical response: 2-5 seconds
  → If slower, check GPU usage: nvidia-smi

"Hermes keeps talking"
  → Ctrl+C to interrupt
  → Type "exit" to quit gracefully

"Agent crashes"
  → Check Python has requests library: pip install requests
  → View error output for details
  → Restart: python hermes_agent.py

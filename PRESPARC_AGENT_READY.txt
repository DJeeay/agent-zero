Your Notion API key is set! Now we need your database ID.

To get it:
1. Open your Notion workspace
2. Go to your database page
3. Look at the URL:
   https://notion.so/xxxxxxxxxxxxxxxxxxxxx?v=yyyyyy
   
4. Copy the long ID (before the ?)
5. Run:
   docker compose -f docker-compose-llm-only.yml exec -e NOTION_DATABASE_ID=xxxxx presparc_agent.py status

Or edit .env:
   NOTION_DATABASE_ID=your_id_here

================================================================================
                        TROUBLESHOOTING
================================================================================

"LLM not responding"
  → Check: curl http://localhost:8080/health
  → Wait: First startup takes 1-2 minutes
  → View logs: docker logs llm-server-presparc

"Notion connection fails"
  → Check .env has correct API key
  → Verify NOTION_DATABASE_ID is set
  → View logs in presparc_agent.py

"Agent crashes"
  → Ensure LLM is healthy first
  → Check Python has requests library: pip install requests
  → View error output for details

"Out of memory"
  → docker stats llm-server-presparc
  → LLM can use 12GB (normal)
  → Reduce --n-gpu-layers if needed

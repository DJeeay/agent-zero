## 📞 Support Rapide

### "Comment vérifier que tout marche?"
```bash
docker ps              # Voir les conteneurs actifs
docker logs -f llama-server  # Voir les logs
docker stats           # Voir la consommation
```

### "Comment monitorer la mémoire?"
```bash
docker inspect llama-server --format '{{.HostConfig.Memory}}'
# Affiche: 12884901888 (= 12GB en bytes)
```

### "Peut-on réactiver llamacpp?"
```
Oui, mais recommandé seulement si:
- Vous avez 2+ GPUs
- Vous assignez GPU 1: docker update --gpus '"device=1"' llamacpp
- Vous testez des configurations différentes
```

### "Où est l'état d'Agent-Zero?"
```bash
docker volume inspect AGENT0Volume
# Stocké sur le disque, préservé lors du nettoyage
```

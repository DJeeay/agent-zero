================================================================================
                        TROUBLESHOOTING
================================================================================

LLM still loading?
  → This is NORMAL. First startup takes 1-2 minutes.
  → Check: curl http://localhost:8080/health
  → Expected response: {"status":"ok"}

Agent not responding?
  → Check LLM is healthy first
  → View logs: docker logs presparc-agent
  → Restart: docker compose -f docker-compose-agent-simple.yml restart presparc-agent

Notion connection failing?
  → Check .env has correct API key
  → Verify key is not empty: cat .env | grep NOTION
  → Check logs for "NOTION_API_KEY" errors

Out of memory?
  → Check usage: docker stats
  → LLM can use up to 12GB (normal)
  → Reduce in docker-compose-agent-simple.yml line 31
